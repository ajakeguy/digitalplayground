Tool Chest

The pace of innovation and development in the world of Bitcoin and Free and Open Source Software is happening at lightning speed. Below are some tools that anyone looking into the space should be familiar with as they are doing their research.

Bitcoin Hardware Wallets / cold storage devices

  • Trezor
  • Bitbox
  • Foundation devices
  • Coinkite (cold card & open dime)


Bitcoin Desktop Wallets

  • Specter Wallet
  • Sparrow Wallet
  • Electrum Wallet


Privacy-centric day-today tools

  • Matrix - An open network for secure, decentralized communication
  • Bitwarden - open-source password manager that you can self-host
  • File Browser - self-hosted file storage (plug & play on Start9Labs Embassy)
  • Proton Mail - encrypted email
  • Proton Calendar - Encrypted calendar that can be synced with Protonmail
  • CalyxOS - "Android mobile operating system that puts privacy and security into the hands of everyday users" for mobile devices
  • GrapheneOS - privacy and security focused mobile OS with Android app compatibility
  • Threema - secure and private messenger
  • Briar - Peer-to-peer encrypted messaging
  • Syncthing - Private, encrypted, open source file sync
  • Pop!_OS - Basic Linux operating system


Bitcoin Nodes (plug & play or DIY)

  • nodl Dojo
  • Embassy by Start9Labs
  • myNode
  • nodl


Bitcoin 'Hot Wallets'

  • Samourai Wallet - privacy centric tech stack
  • muun wallet - self-custodial wallet for bitcoin and lightning


Bitcoin Lightning Applications

  • Mintgox - online gaming built on top of lightning where you can earn sats
  • Muun Wallet- Bitcoin and lightning wallet
  • Shockwallet- "The Lightning Platform for Creators and Entrepreneurs."
  • Thunderhub - "monitor and manage your node from any browser and any device."
  • Zap - Bitcoin lightning wallet
  • ZEBEDEE - Bitcoin gaming wallet
  • Zeus - Bitcoin lightning wallet
  • The Lightning Address- "Like an email address, but for your Bitcoin"
